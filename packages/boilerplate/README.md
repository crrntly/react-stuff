# React Project Starter

Features:

- Transpiling with Babel and PostCSS
- Bundling with Wepack
- Image optimization
- Linting with Eslint and Stylelint
- Hot-reloading development environment
- Testing with Jest

## Usage

Local development:

```
$ npm start
```

Build for production

```
$ npm run build
```

Test:

```
$ npm run lint
$ npm test
```
