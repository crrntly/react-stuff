# `@crrntly/eslint-config-react-app`

An [eslint](https://eslint.org/) configuration for React projects.

## Installation

```sh
npm install --save-dev @crrntly/eslint-config-react-app
```

## Usage

### package.json

```json
{
  "eslintConfig": {
    "extends": [
      "@crrntly/eslint-config",
      "@crrntly/eslint-config-react-app"
    ]
  }
}
```
